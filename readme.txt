Plugin for create your own markers on Google Maps, as a Custom Type posts, and with an easy integration on your page.

== Description ==

Gmap Point List is a plugin that lets you create your own map from Google Maps, with one or more markers, with custom address. Requires jQuery loaded in the top of the Web. Map multiple instances on the same page is not allowed.

== Installation ==

1. Upload the entire `gmap-point-list` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Markers' menu in your WordPress admin panel.
3. First of all, create a map, on submenu 'Map List'. And then copy the shortcode and paste where you need (page/post...).
4. Add markers, on submenu 'New marker'. Please, fill all fields.

5. Maybe, you want to list your markers on a widget. You can see the shortcode on the 'Map List' section.
